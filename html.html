HTML

Toda vez que você acessa um site pela internet você já se perguntou de como é construida aquela página? Bom existe toda uma história por trás de como nós chegamos nesse atual momento da nossa tecnologia que permite que nós a qualquer momento possamos acessar quase qualquer tipo de informação. Para que isso fosse possível algumas invenções e criações foram necessárias , se tratando de páginas web uma delas foi o HTML !

Veremos nesse artigo um pouco da história dessa incrivel tecnologia chamada abreviadamente HTML.

História

HTML (abreviação de HyperText Markup Language, que em português seria "Linguagem de Marcação de Hipertexto") é a principal linguagem de marcação utilizada na construção de páginas na Web. Documentos HTML podem ser interpretados por navegadores.Tim Berners-Lee físico britânico foi o criador do HTML. Tim Berners-Lee, se inspirou numa linguagem chamada SGML para o desenvolvimento do HTML mas no HTML já era possivel ligar uma pagina a outra isso passou a ser chamado de hiperlinks.

A linguagem SGML (Standard Generalized Markup Language) montava documentos compartilhados na Web,algumas tags foram herdadas do SGML para o HTML

O propósito inicial de Tim Berners-Lee sobre a linguagem de marcação HTML era uma solução para o compartilhamento de documentos científicos entre pesquisadores, e para facilitar esse trabalho , ele com a ajuda do estudante Robert Cailliau do CERN (European Council for Nuclear Research-"Conselho Europeu para pesquisa Nuclear"), empenharam-se a criar o HTML.

Em 1989, conseguiu realizar a primeira comunicação entre um cliente HTTP e um servidor e depois tendo sua primeira versão em 1992. Vale a pena lembrar que esse gênio Tim Berners-Lee também criou o protocólo HTTP e a World Wide Web (o famoso www).


Versões do HTML até hoje

Versões

1995 - Versão 2.0
Foi apresentado na primeira conferência da World Wide Web, nessa versão o HTML já possuia sua principais características. Primeira versão que surgiu depois da criação do HTML.
    1999 - Versão 3.2
Já era possível criar tabelas , texto ao redor de imagens etc. Nessa versão houve correções da versão 2.0 principalmente correções de compatibilidade.
    1999 - Versão 4.01
No HTML 4.0 toda a formatação já podia ser transferida do documento HTML para uma folha de estilo separada. Porque a HTML 4.0 separa a apresentação do documento de sua estrutura, total controle do layout de apresentação.
    2014 - Versão HTML 5
Última versão do HTML desenvolvida que aperfeiçou a experiência para os desenvolvedores e usuários. Essa versão possui novos tipos de controles em formulários, como e-mail , pesquisa, data e hora, melhor tratamento de exceção, mais tags para substituir scripts, independência de plataforma e redução da necessidade de plugins externos. o HTML 5 fornece o suporte de áudio e vídeo em alto nível, algo que não existia nas versões anteriores.


Estrutura

O HTML é uma linguagem de marcação, e como outras linguagem é por meio de seu código que o conteúdo é interpretado criando o que vemos numa página. Esses códigos são inseridos em tags, que determinam os elementos presentes na estrutura da página,a fonte da letra, as cores, o seu tamanho , as quebras de linha e etc.












Como pode visto acima a tag <head/> faz parte da estrutura básica do documento, sua finalidade é definir o cabeçalho do documento com informações que não serão exibidas dentro do conteúdo da página.

Depois temos a tag <body/> que vai ser todo o código que vai aprecer na página.

*   Compatibilidade Universal

*   Código mais limpo e de fácil compreensão
